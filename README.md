# YVPN 一个windows pptp l2tp拨号小工具

仅供学习参考

###### 更多信息请访问 [cnygf.net](http://www.cnygf.net).

## 快速开始
#### 下载和编译
    git clone https://github.com/cn-ygf/yvpn.git
    使用vs2013或其它版本打开sln 编译

#### 二进制下载
### [yvpn v3.0.5.beta.zip](https://github.com/cn-ygf/yvpn/releases)
